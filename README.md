# LeagueStats-DiscordBot

A Discord chat bot to keep track of League of Legends stats from match history screenshots.

This project is a work in progress. 

## Prerequisites

* Python 3.14
* asyncio library
* discord library
* OpenCV library
* python-dotenv library
* easyocr library
* numpy library

## Goal
This bot was created with the intention of being used to keep track of player stats in League of Legends games that are not tracked online like 5v5 custom games.
